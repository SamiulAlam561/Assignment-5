What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents the structure of a web document as a tree of objects. It allows scripts to access and manipulate elements, attributes, and text within the document. This enables dynamic updates and interactions, making it essential for creating interactive web applications.

How do select an element from the DOM?

You can select an element from the DOM using methods like getElementById, getElementsByClassName, getElementsByTagName, querySelector, and querySelectorAll.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using setAttribute, getAttribute, and removeAttribute. For styles, use the style property to change CSS directly or classList to add, remove, or toggle classes.

What is event delegation in the context of DOM and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element, allowing it to handle events from child elements that bubble up. This improves performance, simplifies code management, and automatically handles events for dynamically added children.
